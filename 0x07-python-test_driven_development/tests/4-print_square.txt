 How to Use print_square.py
===============================

This library is very simple, since it only has one function called
``print_square()``.

Numbers
=======

``print_square()``  prints a square with the character #.

::
    >>> print_square = __import__('4-print_square').print_square
    >>> print_square(4)
    ####
    ####
    ####
    ####
    >>> print_square(10)
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    ##########
    >>> print_square(0)
