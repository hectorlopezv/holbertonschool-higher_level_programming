===============================
 How to Use add_integer.py
===============================

This library is very simple, since it only has one function called
``add_integer()``.

Numbers
=======

``add_integer()`` returns the sum of its arguments.  For numbers,
that value is equivalent to using the ``+`` operator.

::

    >>> add_integer = __import__('0-add_integer').add_integer
    >>> add_integer(2, 3)
    5

Trying with floats
::
    >>> add_integer(100.3, -2)
    98

Trying with just "a" argument
::
    >>> add_integer(2)
    100

Trying with negative arguments
::
    >>> add_integer(100, -2)
    98

Trying with no arguments
::
    >>> add_integer()
    Traceback (most recent call last):
    ...
    TypeError: add_integer() missing 1 required positional argument: 'a'

Trying with arguments different than float or int
::
    >>> add_integer("a")
    Traceback (most recent call last):
    ...
    TypeError: a must be an integer
    >>> add_integer(1, "g")
    Traceback (most recent call last):
    ...
    TypeError: b must be an integer
    >>> add_integer(None)
    Traceback (most recent call last):
    ...
    TypeError: a must be an integer
